# 晴れノチ（仮名）


## 概要

北海道全体の大まかな天気予報と、GoogleMapの連携
"Open Weather API"より天気予報を取得、目的都市とスポット名を選択し、GoogleMapにて検索する


## 着想　目的
夏季・ライダーなどの「天気が良い場所を走りたい」と言う、非目的地決定型ドライブ利用を想定


## 効果

- 北海道経済の活性化
